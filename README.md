# Gradle로 구성한 Spring Boot 학습용 블로그 프로젝트

## 프로젝트 소개
JPA와 MyBatis를 활용한 블로그 프로젝트입니다.

## 기술 스택
- **Framework**: Spring Boot 3.5.0
- **Build Tool**: Gradle
- **Database**: MySQL 8.0
- **ORM**: JPA + MyBatis

## 주요 기능
- 사용자 관리 (CRUD)
- 게시글 관리 (CRUD)
- 댓글 시스템

## 실행 방법
bash
./gradlew bootRun
